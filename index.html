แนะนำตัว

สวัสดีครับ ผมมีชื่อว่า นาย จักรพรรดิ์ นิทรัพย์ กำลังศึกษาอยู่ในชั้นมัธยมศึกษาที่ 6 ของโรงเรียนอ่างทองปัทมโรจน์วิทยาคม ผมได้ตัดสินใจสร้างเว็บไซต์แบบง่ายๆ นี้ขึ้นมาเพื่อเป็นผลงานที่ใช้แนบไปกับ Portfolio ที่ทุกท่านน่าจะได้อ่านกันแล้ว พูดด้วยความสัตย์จริง ผมเป็นคนนึงในหมู่นักเรียนด้วยกันที่มีความสามารถเฉพาะด้านครับ ผมหลงไหลการเขียนโปรแกรมและเทคนิคทางคอมพิวเตอร์มาตั้งแต่เด็กๆ แล้ว และผมจะสานฝันของผมในรั้วมหาลัยแห่งนี้ ที่มหาวิทยาลัยเทคโนโลยีธัญญบุรี

แน่นอนว่าทุกท่านคงอยากจะทราบว่า อันตัวผมนั้นมีความสามารถในการเป็นนักเรียนสาย FullStack Developer มากแค่ไหน ผมได้แนบเกียรติบัตรที่มีไว้เพื่อพิสูจน์ความรู้ของผมส่วนใหญ่ไว้ใน Portfolio แล้ว ซึ่งนั่นหมายถึง นั้นไม่ใช่เกียรติบัตรทั้งหมดที่ผมมี ผมยังมีความสามารถในการเขียนโค้ดในภาษาอื่นๆ อย่าง Python, C# และ Java แบบเบื้องต้นนอกเหนือจากภาษาที่เกี่ยวกับ Front End ด้วย

ความสามารถที่มีโดยย่อ

HTML 5

ความสามารถในภาษา HTML ของผมนั้น มีในระดับที่สามารถนำไปเขียนเว็บไซต์แบบทั่วๆ ไปได้ มีความรู้ระดับทั่วไปที่ HTML4 มี และมีความรู้ระดับพื้นฐานที่ HTML5 มี

CSS

ความสามารถในภาษา CSS ที่ผมมีนั้น เพียงพอต่อการนำมาสร้างเว็บไซต์ที่สวยงามแบบ Minimal Style มันอาจจะไม่อลังการมาก แต่ก็ทำให้ User ถูกใจได้ง่ายๆ

JS (Java Script)

ภาษานี้เป็นภาษาโปรแกรมที่ออกแบบมาเพื่อเชื่อมต่อกับ HTML เพื่อทำให้เว็บไซต์สามารถ Interactive ได้มากขึ้น ผมมีความรู้พื้นฐานทั่วๆ ไปของภาษานี้และสามารถต่อยอดเพิ่มเติมได้ในระดับมหาวิทยาลัย

Python

Python เป็นภาษาโปรแกรมแรกที่ผมได้เรียน ผมมีความเข้าในและทักษะการเขียนโปรแกรมในภาษานี้สูงมาก โดยผมตั้งใจที่จะนำมันมาใช้เป็นภาษาใน Back End ในงานต่างๆ ในอนาคต

C#

C# เป็นภาษาที่ผมสนใจเพราะว่ามันเป็นภาษาที่ใช้ในการเขียนเกมใน Unity เพราะนอกจาก FullStack Developer แล้ว การพัฒนาเกมก็เป็นอะไรที่ผมหลงใหลเช่นกัน ผมมีความเข้าใจในระดับทั่วๆ ไปของภาษานี้ สามารถศึกษาต่อยอดเพิ่มเติมได้ง่าย และสามารถเขียนได้หากมี Code Snippet

Java

Java นั้นเป็นภาษาที่คล้ายกับ C# ในหลายๆ ด้านผมมีความรู้ทางด้านภาษา Java อยู่พอสมควร สามารถเขียนได้ถ้ามี Code Snippet

PHP

ผมเข้าใจดีว่า ไม่ว่าจะเป็นยุคสมัยไหน ในการทำงานจริง PHP จะไม่มีวันตาย ผมจึงเรียนรู้ Basic ของมันไว้พอสมควร เตรียมพร้อมสำหรับการนำไปต่อยอดในระดับมหาวิทยาลัย

Linux Skill

ผมมีความสามารถในการใช้คอมพิวเตอร์ระบบปฏิบัติการ Linux ตระกูล Debian ทั่วๆ ไป

Git & Github

ผมมีความสามารถเริ่มต้อนในการใช้งาน Git บนลินุกซ์ และเว็บ Github ทำให้สามารถทำงานกับเพื่อนร่วมทีมได้อย่างไม่ยากเย็น
